# Design References

External and cross-project design inputs for this project.

## Vagary-Brand-Guidelines.pdf

The **global Vagary brand philosophy** — a reference and suggestion only.

This project has its own design and branding. It **derives from** (does not copy)
this global philosophy. Treat the guidelines as orientation for tone, values, and
visual sensibility; the project's own `docs/design/` artifacts (`brand.md`,
`palette.md`, `typography.md`, `voice.md`, etc.) remain authoritative for this
product's identity.

If a conflict arises between this PDF and this project's own design docs, the
project's own docs win.
